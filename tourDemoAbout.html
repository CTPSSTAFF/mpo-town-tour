Introduction

This application offers a virtual tour of the 101 cities and towns in the Boston Region MPO. It is primarily intended as a demonstration of some of the experimental animation features of version 3 of the OpenLayers web client mapping library. The data displayed for individual towns is largely identical that displayed in CTPS's Livable Communities Data Browser; a few additional fields were added more-or-less at the whim of the developer.

Software Used

The software used to build this demo includes:
  • PostgreSQL - database
  • PostGIS - a spatial extension to the PostgreSQL database
  • Geoserver - to publish spatial data (in this case, city and town boundaries) and tabular data on the web
  • OpenLayers (version 3) - for mapping functionality in the web client
  • jQuery - to help manage the DOM

Data Sources

The base map displayed by this demo is Open Street Map.

The tabular data displayed by this demo come from:

  • United States Bureau of the Census
  • Massachusetts Office of Geographic Information (MassGIS)
  • Massachusetts Department of Transportation (MassDOT)
  • Boston Region Metropolitan Planning Organization
  • Metropolitan Area Planning Council (MAPC)
